CPS tester is a fun tool that tracks your CPS, clicks per second.

CPS tester has a history to see your past 10 runs.

## Prerequisites

- Tkinter
- sqlite3

## Usage

1. Run `python cps_tester.py`
2. Start testing your CPS by clicking
